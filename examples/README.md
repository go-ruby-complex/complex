# Ruby examples

Pure-Ruby examples for `Complex`, provided by
[go-embedded-ruby](https://github.com/go-embedded-ruby/ruby) (rbgo). Complex is
a core part of the numeric tower, so no `require` is needed. Run them with the
`rbgo` interpreter:

```sh
rbgo examples/complex_usage.rb
```

| File | Shows |
| --- | --- |
| [`complex_usage.rb`](complex_usage.rb) | Building `Complex`, exact-preserving arithmetic, `abs`/`conjugate`. |

Each example is executed as-is under rbgo.
